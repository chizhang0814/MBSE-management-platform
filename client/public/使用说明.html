1注册与登录

注册

  1. 打开平台，点击登录页下方 「没有账号？注册新账号」
  2. 填写用户名（3-20位字符，注册后不可更改）和密码（至少6位）
  3. 注册成功后自动跳转登录，此时账号无任何项目权限

登录

输入用户名和密码，登录后右上角显示用户名及当前所有项目角色。

2个人设置

点击右上角 齿轮图标 打开个人设置。

修改姓名 / 部门

在"基本信息"区域直接编辑，点击 保存 生效。

修改密码

填写当前密码和新密码（至少6位），点击 保存。

申请项目权限

  1. 在"申请项目权限"选择项目和角色，点击 申请
  2. 系统提示提交成功后，该项目的总体人员会收到通知进行审批
  3. 审批通过后，刷新页面权限即时生效
同一时间只能有一个待审批申请，上一个处理完后才能再次申请。
角色主要权限
总体人员审批变更、审批权限申请
EWIS管理员创建 / 编辑 / 删除信号
设备管理员创建 / 编辑 / 删除设备、连接器、针孔、信号
一级包长只读查看
二级包长只读查看
只读只读查看

3通知中心

点击右上角 铃铛图标，红点表示有未读消息。打开后所有通知自动标为已读。

通知类型收到人需要操作
权限申请 总体人员 点击 通过 或 拒绝
待完善相关设备负责人进入数据页补全字段
待审批总体人员进入数据页审批变更
申请结果申请人无需操作，查看结果即可

4项目数据

5设备管理

需要 设备管理员 角色，且只能操作自己负责的设备。

新建设备

  1. 点击 「新增设备」，填写设备编号（必填）、名称、型号等信息
  2. 选择设备负责人（支持按姓名或工号搜索）
  3. 选择提交方式：
    • 保存草稿：仅保存，不触发审批
    • 提交审批：立即进入审批流程，相关人员收到通知

编辑设备

  1. 点击设备行展开详情
  2. 修改字段后点击 提交变更，自动进入审批流程
  3. 审批通过前设备状态显示为 Pending

管理连接器与针孔

在设备详情展开区域可新增、编辑、删除该设备下的连接器和针孔，操作同样触发审批流程。

6信号管理

创建 / 编辑 / 删除需要 EWIS管理员 或 设备管理员 角色。

新建信号

  1. 点击 「新增信号」，选择连接类型（1to1 / 网络 / ERN）
  2. 填写信号属性（ATA编码、推荐导线规格、电磁兼容代码等）
  3. 配置端点：指定 设备 → 连接器 → 针孔 的映射关系
  4. 提交后进入审批流程

端点确认状态

状态含义
已确认所有端点均已确认
待确认仍有端点未确认

7审批流程

平台所有数据变更（新建 / 编辑 / 删除）均需经过两阶段审批：

1
提交变更 设备管理员 / EWIS管理员 提交新增、修改或删除操作
2
完善阶段 涉及设备的负责人收到通知，补全缺失的必填字段后提交完善
3
审批阶段 项目所有总体人员收到通知，逐一审批（批准 / 拒绝）
✓
结果 全部通过 → 数据生效  |  任一拒绝 → 退回草稿
任意一名总体人员拒绝，整个变更请求即被拒绝。

我需要完善时

  1. 收到"待完善"通知
  2. 进入"项目数据"，切换到 待我完善 筛选
  3. 展开对应数据行 → 填写标红的缺失字段 → 点击 提交完善

我需要审批时（总体人员）

  1. 收到"待审批"通知
  2. 进入"项目数据"，切换到 待我审批 筛选
  3. 展开数据行 → 查看变更内容 → 点击 批准 或 拒绝

查看历史记录

展开任意数据行，点击 「历史」 标签，可查看该条数据的全部审批历史。

8常见问题

Q：看不到某个项目怎么办？
在"个人设置"中申请该项目权限，等待总体人员审批通过即可。
Q：为什么保存后设备变成了 Pending？
关键字段修改会触发审批流程，等待总体人员审批通过后数据才正式生效。
Q：申请权限后等了很久没有审批？
联系该项目的总体人员，请其登录系统查看通知铃铛。
Q：被拒绝后如何重新申请？
申请被处理后可在"个人设置"中重新提交新的申请。
Q：可以同时拥有多个项目的权限吗？
可以，但每个项目只能拥有一个角色。
