# 信号 Unique ID 自动命名规则

## 格式

```
N{ATA_A前2位}{4位序号_A}-{ATA_B前2位}{4位序号_B}[-子编号]
```

## 序号逻辑

- `序号_A`：项目内所有信号 unique_id 中 `N{ATA_A前2位}????` 和 `-{ATA_A前2位}????` 出现的最大 4 位数字 +1
- `序号_B`：项目内所有信号 unique_id 中 `N{ATA_B前2位}????` 和 `-{ATA_B前2位}????` 出现的最大 4 位数字 +1
- 两侧序号各自独立递增，不同步
- 序号在保存时生成（非编辑时），防止并发冲突

## ATA 提取规则

| 端点情况 | ATA 取值 |
|----------|---------|
| 正常设备（如 ATA=23-10） | 取前 2 位 → `23` |
| ERN 设备（8800G0000） | 取另一端的 ATA |
| ATA 为空/N/A/非数字 | 用 `00` |
| 两端都异常 | 两侧都用 `00` |

## 子编号

| 连接类型 | 子编号 |
|----------|--------|
| 普通 1to1 / Discrete | 无 |
| ARINC 429 | `-1`（Positive）、`-2`（Negative） |
| CAN Bus | `-1`（High）、`-2`（Low）、`-3`（Gnd） |
| RS-422 | `-1`（A）、`-2`（B）、`-3`（Gnd） |
| RS-485 | `-1`（A）、`-2`（B）、`-3`（Gnd） |
| 其他协议 | 无 |

## 权限

- **admin**：可手动编辑 Unique ID
- **其他角色**：不可编辑，由系统自动生成

## 生成时机

- 在信号**保存时**（后端 POST/PUT）生成，不在前端编辑时生成
- 防止并发：两个用户同时创建时，序号在 INSERT 前即时查询最大值
